## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/AbdulManer25/My-portfolio.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd My-portfolio
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run the Project

```bash
npm start
```

The app will run on:

```bash
http://localhost:3000
```
